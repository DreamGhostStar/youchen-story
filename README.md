这是一个基于React的UI组件库

在开始项目之前，我们需要先安装并且引入部分文件

#### 安装

```
npm install youchen-story --save
```

#### 引入css文件

我们将组件库的样式文件全部打包为一个css文件，在使用之前，请先在合适的地方引入css文件

```
import 'youchen-story/dist/index.css'
```
